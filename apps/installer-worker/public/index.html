Kaline Offline

Instale sua presença local com segurança.

Este portal identifica seu sistema e entrega o instalador correto da Kaline Offline. A instalação real acontece no seu computador, por scripts locais do próprio repositório.

Linux Mint Xfce

Instalador Linux Mint

Use este instalador no Linux Mint Xfce. Ele baixa um pequeno bootstrap público, localiza ou clona o repositório da Kaline e chama o instalador real já existente em scripts/install-kaline-mint.sh.

Baixar instalador Linux Mint
Comando revisável (recomendado)
curl -fsSL ORIGEM/install/linux-mint.sh -o install-kaline.sh && less install-kaline.sh && bash install-kaline.sh
Comando rápido (sem revisar antes)
curl -fsSL ORIGEM/install/linux-mint.sh -o install-kaline.sh && bash install-kaline.sh
Windows

Instalador Windows

Use este instalador no Windows. Ele baixa um pequeno bootstrap público, localiza ou clona o repositório da Kaline e chama o instalador real já existente em scripts/install-kaline-windows.ps1.

Baixar instalador Windows
Comando PowerShell
irm ORIGEM/install/windows.ps1 -OutFile install-kaline.ps1; powershell -ExecutionPolicy Bypass -File .\install-kaline.ps1

Não conseguimos identificar seu sistema automaticamente — escolha o instalador correto acima (Linux Mint Xfce ou Windows).

O que o instalador fará

Tudo isso acontece localmente, no seu computador — nunca no Worker.

  • Verifica se a Kaline já está baixada
  • Clona Tonyus-dev/kalineatelier se necessário
  • Verifica GitHub CLI (gh)
  • Orienta gh auth login se necessário
  • Chama o instalador real do repositório
  • Instala dependências do projeto
  • Configura .env com backup
  • Verifica Ollama
  • Verifica modelos locais
  • Baixa modelos ausentes apenas com sua confirmação
  • Verifica Whisper Small
  • Verifica Kokoro/Dora
  • Prepara o local-server
  • Inicializa o banco SQLite pelo próprio local-server
  • Valida tabelas locais quando possível
  • Cria atalhos Kaline Offline e Kaline Janelinha
  • No Linux Mint Xfce, tenta fixar a Janelinha na barra com segurança
  • No Windows, cria atalhos na Área de Trabalho e no Menu Iniciar

Segurança

  • O portal não instala nada sozinho
  • O portal não lê seus arquivos
  • O portal não cria atalhos diretamente
  • O portal não recebe áudio
  • O portal não acessa sua Kaline local
  • O portal não pede token GitHub
  • O portal não guarda chaves
  • A instalação real acontece localmente, no seu computador
  • A Kaline Offline roda em 127.0.0.1:64113
  • O microfone só liga quando você clica
⚠ Nunca cole tokens ou chaves secretas neste site.
